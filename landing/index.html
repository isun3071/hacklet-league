# the format

Hackathon, but minutes instead of hours — with a cheering audience. Players get 24 minutes and a single mid-tier AI model to build, document, and defend a web application. No web search. No second AI tool. No copy-paste from outside. Just judgment, prompting fluency, and nerve.

When the clock hits zero, the network cuts and the code freezes — partial AI responses roll back, and an automatic commit captures exactly what existed at the buzzer. Then the real test begins.
